# Push — Physical Blocking

::: info Environment
- **PettingZoo**: `simple_push_v3` | **Agents**: 1 Adversary + 1 Good
:::

## Objective
- **Good Agent**: Sprint to the true target (one decoy landmark exists)
- **Adversary**: Block Good Agent; infer which landmark is real

**Key**: Adversary has higher mass — can physically push Good Agent.

## Run
```bash
python push.py
```

本项目包含 9 个经典多智能体博弈场景，覆盖导航、协作、对抗、通信、欺骗等多种范式。

| 1 | [Simple](/zh/games/simple) | 导航 | 1 | 单智能体导航至地标 | ✗ | 5 |

## 环境分类

### 🏃 导航类
- **Simple**: 最简单的基线环境，测试基本的空间推理

### 🤝 协作类
- **Spread**: 需要多智能体协调分配地标

### ⚔️ 对抗类
- **Adversary**: 信息不对称 + 欺骗策略
- **Push**: 物理对抗 + 质量不对称
- **Tag**: 速度不对称 + 合作追捕

### 📡 通信类
- **Crypto**: 加密/解密/窃听三方博弈
- **Reference**: 双向通信 + 目标推理
- **Speaker-Listener**: 单向通信 + 语义对齐

### 🌍 综合类
- **World Comm**: 大规模团队协调 + 通信广播

## 通用物理参数

所有环境共享以下基础物理参数：

| 参数 | 值 | 含义 |
|:-----|:--:|:-----|
| 时间步 `dt` | 0.1 | 每步的物理仿真时间 |
| 阻尼 `damping` | 0.25 | 速度衰减系数：`v ← v × 0.75` |
| 默认质量 | 1.0 | 智能体默认质量 |
| 灵敏度 `sensitivity` | 5.0 | 动作到力的映射系数 |
| 地图范围 | [-1, 1]² | X、Y 轴取值范围 |

## 通用动作空间

大部分环境使用 5 维连续动作向量：

| 索引 | 含义 | 取值范围 |
|:----:|:-----|:--------:|
| `a[0]` | 无操作 (No-Op) | [0, 1] |
| `a[1]` | 左推力 (Left) | [0, 1] |
| `a[2]` | 右推力 (Right) | [0, 1] |
| `a[3]` | 下推力 (Down) | [0, 1] |
| `a[4]` | 上推力 (Up) | [0, 1] |

净力计算：
- **X 轴**: `F_x = (a[2] - a[1]) × sensitivity`
- **Y 轴**: `F_y = (a[4] - a[3]) × sensitivity`

::: warning 重要提示
- 不要同时激活对立方向（如同时设 left 和 right > 0），否则力会互相抵消
- 接近目标时应减小推力或设为 0 以避免过冲
- 接近边界（|x| 或 |y| > 0.9）时应向中心推
:::
